# 단골만들기 Codex MVP Kit

Codex에 업로드해 단골만들기 1차 MVP 구현 기준으로 사용할 파일만 추린 세트입니다.

## 포함 파일

- `CODEX.md`: Codex용 운영 지침
- `docs/brief.md`: 단골만들기 1차 MVP 최종 스펙
- `.codex/rules/`: Next.js, Supabase, 디자인, QA, 배포 규칙
- `.codex/skills/`: 도메인 정리, 계획 수립, 구현, Supabase 작업 Skill
- `.codex/agents/`: 조사, 프론트 리뷰, Supabase 리뷰, 릴리즈 리뷰 Agent

## 시작 방법

1. ZIP을 Codex 작업 프로젝트에 업로드하거나 압축 해제합니다.
2. Codex에게 `CODEX.md`와 `docs/brief.md`를 먼저 읽게 합니다.
3. `codex-start-prompt.md` 내용을 붙여넣고 시작합니다.

## 핵심 주의

카카오톡 자동발송은 1차 MVP의 핵심 기능입니다.
쿠폰, 링크추적, 고급통계, POS 연동, VIP 태그는 1차에서 제외합니다.
